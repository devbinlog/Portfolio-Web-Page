My Portfolio World

## 주인공

Taebin Kim (김태빈)

## 프로젝트 정의

본 프로젝트는 Taebin Kim의 실명 중심 포트폴리오 플랫폼이다.
단순한 포트폴리오 템플릿이 아니라, 프로젝트들이 탐색 가능한 오브젝트로 존재하는 3D 인터랙티브 개인 세계이다.

이 플랫폼은 다음을 동시에 표현한다.

- 개인 아이덴티티: Taebin Kim이라는 이름을 중심으로 한 명확한 자기소개
- 엔지니어링 역량: 구조적 설계, 배포 가능한 풀스택 아키텍처, 3D 인터랙션 능력
- 프로덕트 사고: 사용자 경험, 정보 구조, 유지보수성, 확장 가능성을 함께 고려

## 핵심 사용자 경험

사용자가 이 포트폴리오를 방문했을 때의 경험 순서:

1. 화면이 열리면 Taebin Kim의 이름과 역할이 먼저 보인다
2. 3D 세계가 펼쳐지며 프로젝트 오브젝트들이 주변에 부유한다
3. 사용자는 오브젝트에 마우스를 올려 프로젝트 이름을 확인한다
4. 오브젝트를 클릭하면 카메라가 해당 프로젝트로 이동하고 상세 정보 overlay가 열린다
5. overlay에서 설명, 이미지, 영상 플레이스홀더, 문서 플레이스홀더, 기술 스택, 링크를 확인한다
6. 선택적으로 독립 상세 페이지(라우트 기반)로 이동하여 깊이 읽기가 가능하다
7. 세계로 돌아가 다른 프로젝트를 탐색한다

이 경험의 핵심 원칙:
- 3D는 공간, 발견, 분위기, 인터랙션을 담당한다
- HTML은 가독성, 밀도 있는 정보, 미디어, 링크를 담당한다

## 세계관 및 포지셔닝

### 세계관 구조

이 포트폴리오는 하나의 개인 세계이다.
세계의 중심에는 Taebin Kim의 아이덴티티 코어가 존재하고,
주변에 프로젝트들이 탐색 가능한 오브젝트로 떠 있다.

세계 계층:
- Identity Core: 개인 아이덴티티의 중심 오브젝트
- Featured Orbit: 대표 프로젝트들의 주요 궤도
- Secondary Orbit: 추가 프로젝트들의 보조 궤도
- Ambient Depth Objects: 분위기를 위한 배경 오브젝트
- Overlay UI Layer: HTML 기반 상세 정보 레이어
- Navigation/Utility Layer: 내비게이션, 테마 전환, 유틸리티

### 시각적 무드

기본 무드: Refined Dark Exhibition (정제된 다크 전시)
보조 지원: Minimal Light Gallery Mode (미니멀 라이트 갤러리)

무드 원칙:
- 몰입감 있되 게임적이지 않다
- 시각적 소음 없이 정보 계층이 명확하다
- 영화적이되 절제되어 있다
- 두 테마 모두 premium하고 calm하다

## 차별점

일반 포트폴리오와의 차별점:

구조적 설계: 기능 구현 전에 문서, 데이터 구조, 아키텍처를 정의한다
3D 인터랙션: 프로젝트가 평면 카드가 아닌 3D 공간의 오브젝트로 존재한다
풀스택 아키텍처: Next.js + NestJS + PostgreSQL + Redis의 실제 운영 가능한 구조
배포 가능성: Docker, 환경 분리, CI/CD, 마이그레이션 전략을 포함
콘텐츠 모델: 영상/문서/보고서가 실제 업로드 전에도 플레이스홀더로 구조화
다크/라이트 테마: 두 모드 모두 1등급 기능으로 설계
관리자 시스템: 프로젝트 CRUD, 미디어/문서 관리, 연락 메시지 확인 가능
일하는 방식 반영: About 섹션에 개발 철학과 작업 방식이 명시적으로 포함
